
# Eggactyl

A multi-egg for the [Pterodactyl Game Panel](https://pterodactyl.io) which can run multiple types of game and voice servers and Discord bots, including:
- Minecraft Java
- Minecraft Bedrock
- Minecraft Proxies
- NodeJS
- Python
- PHP
- TeamSpeak3
And more.
